## 常用请求方法
**请求方法：** 对服务器==资源==，要执行的==操作==

| 请求方法     | 操作       |
| -------- | -------- |
| ==GET==  | ==获取数据== |
| ==POST== | ==提交数据== |
| PUT      | 修改数据（全部） |
| DELETE   | 删除数据     |
| PATCH    | 修改数据（部分） |
**数据提交**：当场景需要在服务器上==保存==
## ==axios请求配置==
- url：请求的URL网址
- ==method==：请求的方法，==GET==可省略
- ==data==：提交的数据
